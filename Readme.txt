Платформа ASP.NET MVC 5

5. Гостиница «Royal Palace Inn» Небольшая гостиница расположена на берегу озера. 
 - Для бронирования номеров требуется позвонить администратору по телефону. 
 - Клиенты могут быть размещены в 1-,2-х и 3-х местных номерах категорий стандарт, комфорт и люкс.
 - По желанию можно поставить детскую кровать в любой из номеров. 
 - На разные дни недели стоимость номеров различается. 
 - При бронировании номера на длительное время предусмотрена система скидок.

- Дневник по практике(Подойдет Kanban, можно добавить git log --pretty=oneline --graph)
- Отчет по практике(По-любому делает Никитос)
